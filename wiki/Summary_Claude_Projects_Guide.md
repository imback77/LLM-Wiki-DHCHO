# Summary_Claude_Projects_Guide

## 주요 내용
- **[[Claude Projects]]의 기능**: 사용자별 지식 베이스(Project Knowledge)와 사용자 정의 지침(Custom Instructions)을 결합하여 컨텍스트가 유지되는 전용 작업 공간을 제공함.
- **기술적 특징**:
  - **RAG 모드 자동 전환**: 업로드된 문서가 컨텍스트 윈도우 한계를 초과할 경우 Anthropic의 자체 RAG 시스템이 자동으로 활성화됨.
  - **메모리 관리**: 프로젝트별로 독립된 메모리 요약을 유지하여 정보의 간섭을 방지.
- **사용자별 제약**: 무료 사용자는 최대 5개 프로젝트 생성 가능, 유료(Pro/Team)는 더 넓은 용량과 공유 기능 제공.

## 핵심 인사이트
- **프로젝트 기반 지식 관리**: 단순 채팅을 넘어 "정보의 축적과 재사용"이 Claude의 핵심 사용자 경험으로 자리 잡음.
- **에이전틱 워크플로**: 프로젝트 지침을 통해 에이전트의 페르소나와 작업 스타일을 고정시켜 반복적인 설정 없이 일관된 결과 도출 가능.
