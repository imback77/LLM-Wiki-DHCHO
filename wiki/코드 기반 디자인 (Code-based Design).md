# 코드 기반 디자인 (Code-based Design)

> 디자인 요소를 픽셀 기반의 그래픽 툴이 아니라, 정의된 코드 토큰(타이포그래피, 컬러, 그리드) 기반의 시스템으로 처리하는 패러다임.

## 주요 장점
- 디자이너가 아닌 비전문가/개발자도 수준 높은 결과물을 즉시 배포 가능.
- **Claude Code**와 같은 AI 코딩 에이전트에 지시할 때 오류율이 적고 체계적인 확장이 가능함.
- `awesome-design`과 같은 오픈소스 브랜드 디자인 시스템을 적용하여 단숨에 고품질 룩앤필 확보.
