# Gemini (제미나이)

> 구글(Google)이 개발한 최첨단 멀티모달 생성 AI 모델 시리즈. 텍스트, 이미지, 오디오, 비디오, 코드를 동시에 이해하고 생성할 수 있는 범용 능력을 갖춤.

---

## 최신 정보 (2026-04-21 기준)

- **제미나이 3.1 (Gemini 3.1)**: 성능과 추론 능력이 극대화된 최신 버전. 구글 크롬(Chrome) 브라우저에 직접 통합되어 국내 출시됨.
- **브라우저 통합**: 별도 웹사이트 접속 없이 크롬 사이드 패널을 통해 웹 콘텐츠 요약, 질의응답, 크로스 탭 데이터 병합을 수행.
- **모바일 에이전트**: 안드로이드 OS와 시스템 레벨에서 결합되어 제스처(전원 버튼 길게 누르기)로 현재 화면을 즉시 분석.

## 주요 기능

- **컨텍스트 기반 질의응답**: 현재 사용자가 보고 있는 웹 페이지나 앱 화면의 맥락을 정확히 이해.
- **구글 에코시스템 연동**: 지메일(Gmail), 구글 캘린더, 지도, 유튜브 등과 실시간 데이터 연동.
- **멀티모달 처리**: [[나노_바나나_2]] 등 특화 모델과 협업하여 이미지 생성 및 수정을 브라우저 내에서 지원.
