# Ollama (올라마)

## 📌 개요
Llama, Gemma, Mistral 등 다양한 오픈소스 LLM을 로컬이나 클라우드 환경에서 매우 쉽고 빠르게 실행할 수 있도록 돕는 오픈소스 프레임워크입니다.

## 🚀 주요 특징
- **간편한 실행**: 단일 명령어로 모델을 `pull`하고 서빙을 시작할 수 있습니다.
- **컨테이너 친화적**: 모델 가중치를 Docker 이미지 내에 포함하기 쉬워 서버리스 환경(Cloud Run 등)의 PoC 및 초기 프로젝트 설계에 유리합니다.
- **로컬 에이전틱 서버**: 로컬 환경에서 개인용 AI 에이전트를 구축할 때 표준 도구로 활용됩니다.

## 🛠️ 활용 사례
- **[[Cloud Run]] 배포**: 모델을 컨테이너 내부에 번들링하여 빠른 Cold Start 성능을 확보하는 방식에 사용됩니다. (관련: [[2026-04-20_Gemma_4_Cloud_Run_자가_호스팅_가이드]])
