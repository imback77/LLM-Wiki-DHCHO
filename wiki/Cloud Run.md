# Cloud Run

## 📌 개요
Google Cloud의 서버리스 컴퓨팅 플랫폼으로, 컨테이너화된 애플리케이션을 자동으로 확장 및 축소하며 실행합니다. 최근 GPU 지원(NVIDIA L4 등)이 추가되어 LLM 서빙의 핵심 인프라로 부상했습니다.

## 🚀 주요 장점 (AI 서빙 관점)
- **서버리스 GPU**: 사용한 만큼만 비용을 지불하면서 고성능 GPU 자원을 LLM 추론에 활용할 수 있습니다.
- **자동 확장**: 트래픽에 따라 인스턴스를 자동으로 조절하여 운영 부담을 최소화합니다.
- **스토리지 결합**: GCS Fuse를 통해 대용량 모델 가중치 파일을 로컬 경로로 마운트하여 효율적인 로딩이 가능합니다.

## 🛠️ 활용 사례
- **[[Gemma 4 (젬마 4)]] 자가 호스팅**: Ollama 또는 vLLM 엔진과 결합하여 오픈 모델을 독립적인 서비스로 배포하는 데 사용됩니다. (관련: [[2026-04-20_Gemma_4_Cloud_Run_자가_호스팅_가이드]])
