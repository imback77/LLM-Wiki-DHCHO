# vLLM

## 📌 개요
LLM 추론 및 서빙의 처리량(Throughput)을 극대화하기 위해 설계된 오픈소스 라이브러리입니다. 특히 메모리 관리 기술을 통해 대규모 프로덕션 서비스 환경에서 표준적으로 사용됩니다.

## 🚀 주요 특징
- **Page Attention**: OS의 가상 메모리 관리 기법에서 영감을 얻은 KV 캐시 관리 기술로, 메모리 낭비를 줄여 처리량을 수배 이상 높입니다.
- **연속 배치 (Continuous Batching)**: 들어오는 요청을 동적으로 배치 처리하여 지연 시간을 줄입니다.
- **분리형 스토리지 운영**: 모델 가중치를 엔진과 분리하여 운영할 수 있어 유연한 배포가 가능합니다. (예: GCS Fuse 마운트 방식)

## 🛠️ 활용 사례
- **엔터프라이즈 급 배포**: 모델 가중치를 클라우드 스토리지에 보관하고 서빙 엔진만 독립적으로 운영하는 고성능 배포 아키텍처에 활용됩니다. (관련: [[2026-04-20_Gemma_4_Cloud_Run_자가_호스팅_가이드]])
