# 구글 크롬 (Google Chrome)

> 세계에서 가장 널리 사용되는 웹 브라우저. 2026년부터 본격적으로 'AI 네이티브 브라우저'로 진화하며 운영체제와 클라우드 AI의 가교 역할을 수행함.

---

## AI 주요 기능 (제미나이 통합)

- **사이드 패널 AI**: [[Gemini]]를 사이드바 형태로 상주시켜 웹 서핑 흐름을 방해하지 않고 정보를 분석.
- **크로스 탭 합성 (Cross-tab Synthesis)**: 여러 탭에 흩어진 정보를 하나의 표나 목록으로 자동 요약 및 비교.
- **이미지 생성·편집**: [[나노_바나나_2]] 모델을 통해 브라우저 내에서 즉각적인 배경 제거나 이미지 생성을 지원.
- **AI 사이드바 (Side Panel)**: 브라우저 메뉴 옆의 통합 아이콘을 통해 활성화.

## 플랫폼으로서의 변화

- 단순한 웹 뷰어에서 벗어나 고성능 AI 모델([[2026-04-21_구글_제미나이_인_크롬_국내_출시_분석]])을 직접 탑재함으로써 복작한 인지 업무를 브라우저 내에서 처리.
- 구글 워크스페이스(지메일, 캘린더, 문서 등)와의 연동성 극대화.
