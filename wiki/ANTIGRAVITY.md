# wiki/ — AI가 컴파일하는 지식 위키

> 이 폴더는 AI가 소유하고 유지하는 공간입니다.
> raw/ 폴더의 원본 자료를 읽고, 요약·연결·종합하여 구조화된 위키 페이지를 만듭니다.

## 핵심 원칙
- AI가 이 폴더의 모든 파일을 **작성·수정·삭제**한다
- 후니님은 **읽기 전용** — 읽고 피드백을 주되 직접 편집하지 않는다
- 모든 페이지는 `[[위키링크]]`로 상호 연결한다

## 페이지 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **개념(Concept)** | 하나의 주제에 대한 정리 | `프롬프트 엔지니어링.md` |
| **엔티티(Entity)** | 특정 인물·제품·서비스 | `ChatGPT.md`, `워런 버핏.md` |
| **요약(Summary)** | 원본 자료 1개의 핵심 요약 | `요약_2026-04-16_기사제목.md` |
| **비교(Comparison)** | 두 개 이상의 대상 비교 | `비교_Claude vs ChatGPT.md` |
| **인사이트(Insight)** | 질문·탐색에서 나온 새 발견 | `인사이트_AI시대 중장년 전략.md` |

## 필수 파일
- `index.md` — 전체 위키의 목차 (카테고리별 페이지 목록)
- `log.md` — 위키 활동 기록 (시간순, append-only)

## 페이지 템플릿 (YAML frontmatter)
```yaml
---
type: concept | entity | summary | comparison | insight
created: YYYY-MM-DD
updated: YYYY-MM-DD
sources: [원본 파일 경로]
tags: [태그1, 태그2]
---
```

## AI 작업 지침
- 새 페이지 생성 시 반드시 `index.md`를 업데이트하라
- 모든 변경 사항을 `log.md`에 기록하라
- 기존 페이지와 관련 있으면 `[[위키링크]]`로 연결하라
- 모순이 발견되면 명시적으로 표기하라: `> [!WARNING] 모순 발견: ...`
